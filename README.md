# Executor
A discord bot for executing code from discord!

# About

Executor is a discord bot to run code from various programming languages! starting from python, java, bash and more!

# Supported Languages

- Java
- Bash
- Python
- Elixir
- Typescript
- Php
- Rust
- Lua
- Go
- Erlang
- Lolcode

More will be added soon!
